# 爆款智坊 - AI辅助内容创作培训平台

> 面向证券投顾的AI创作培训与资源网站，提供系统化的AI工具使用培训、Skill工具下载和内容创作工作流

## ✨ 项目简介

**爆款智坊**是一个**培训投顾使用AI辅助创作的网站项目**，核心目标是帮助投资顾问掌握使用AI工具进行新媒体内容创作的技能。

### 核心价值

- 📚 **系统化培训**：在线AI创作培训课程
- 🛠️ **Skill工具**：现成的Claude Code Skill工具下载
- 📝 **提示词库**：6种账号风格的系统提示词
- 💡 **完整工作流**：从初稿到配图的全流程方案

## 🌐 在线体验
