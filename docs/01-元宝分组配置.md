# 元宝分组配置指南

## 概述

元宝的"对话分组"功能是实现专业AI分身的绝佳工具。通过创建高达6000字符的系统提示词分组，可以固化你的专业人格和创作规范。

## 配置步骤

### 1. 创建分组

1. 打开元宝应用
2. 点击"新建分组"
3. 命名分组，例如："申万宏源-小红书投教"

### 2. 编写系统提示词

一个完整的系统提示词应包含以下模块：

#### 角色定义
```
你是一位资深的证券投资顾问，拥有10年以上的从业经验...
```

#### 任务描述
```
你的主要工作是：
1. 为投资者提供专业的投资教育内容
2. 解读市场热点和投资机会
3. 普及金融知识和投资理念
```

#### 目标受众
```
你的内容面向：
- 新手投资者：需要基础知识和入门指导
- 进阶投资者：需要深度分析和策略建议
```

#### 工作要求
```
内容创作要求：
1. 专业准确：所有数据和观点必须有可靠来源
2. 通俗易懂：避免过多专业术语，用生活化语言解释
3. 实用性强：提供可操作的建议和思路
4. 风险提示：明确提示投资风险
```

#### 创作流程
```
在创作内容时，请遵循以下步骤：
1. 理解用户需求
2. 检索相关知识
3. 构建内容框架
4. 撰写具体内容
5. 添加风险提示
6. 格式化输出
```

#### 输出格式
```
小红书笔记格式：
- 标题：吸引眼球，突出核心价值
- 正文：分段清晰，重点突出
- 表情：适当使用emoji增强可读性
- 标签：添加相关话题标签
```

#### 参考案例
```
以下是一个优秀案例，请参考其风格和结构：

【标题】
基金定投3年，我终于赚到了第一桶金！🎉

【正文】
...
```

### 3. 粘贴系统提示词

1. 将编写好的完整系统提示词复制
2. 粘贴到分组的"系统指令"区域
3. 点击保存

## 使用技巧

### 场景化分组

为不同创作场景创建专门分组：
- 投教笔记分组
- 市场解读分组
- 产品推广分组
- 风险提示分组

### 持续优化

1. 收集AI生成效果
2. 识别不足之处
3. 迭代优化提示词
4. 测试验证效果

## 注意事项

1. **字符限制**：元宝支持最多6000字符，请合理分配
2. **定期备份**：重要提示词建议备份到本地
3. **版本管理**：记录每次修改的时间和原因
4. **测试验证**：每次修改后进行多轮测试

## 进阶用法

### A/B测试

创建两个相似分组，使用不同提示词，对比效果：

```
分组A：强调专业性和数据
分组B：强调故事性和情感共鸣
```

### 风格迁移

使用"元提示词"分析爆款内容，自动提取风格要素：

```
请分析以下小红书爆款笔记的风格特征：
1. 人格与口吻
2. 开头策略
3. 结构框架
4. 语言风格
5. 情绪曲线
6. 结尾设计
```

## 常见问题

**Q: 为什么我的分组效果不稳定？**

A: 检查以下几点：
- 系统提示词是否足够详细
- 是否有明确的输出格式要求
- 是否提供了足够的参考案例

**Q: 如何复制分组到另一个账号？**

A: 导出系统提示词文本，在新账号创建相同分组并粘贴。

**Q: 一个分组可以用于多个场景吗？**

A: 建议为每个主要场景创建专门分组，效果更佳。
