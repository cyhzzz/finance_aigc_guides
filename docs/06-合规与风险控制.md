# 合规与风险控制指南

## 概述

作为证券从业人员，AI辅助创作必须严格遵守合规要求，确保内容真实、准确、完整，充分揭示风险，保护投资者合法权益。

## 核心原则

### 原则1：真实性原则

**要求**：所有信息必须真实可靠

```
✅ 正确做法：
- 数据引用注明来源
- 观点基于事实
- 不编造不存在的信息
- 不夸大投资收益

❌ 禁止行为：
- 虚构数据
- 杜撰专家观点
- 夸大投资收益
- 隐瞒重要事实
```

**AI幻觉防范**：

```
AI可能一本正经地胡说八道：
"某基金2023年收益率150%"（实际只有15%）

应对措施：
1. 无条件执行事实核验
2. 所有数据查证可靠来源
3. 关键信息必须人工复核
4. 保留核验记录备查
```

### 原则2：适当性原则

**要求**：内容适合目标投资者

```
风险评估矩阵：

低风险产品：
- 货币基金、国债
- 适合：所有投资者
- 提示：流动性风险

中风险产品：
- 债券基金、蓝筹股
- 适合：有一定风险承受能力
- 提示：市场波动风险

高风险产品：
- 股票基金、期货
- 适合：风险承受能力强
- 提示：本金损失风险
```

**内容匹配**：

```
新手投资者内容：
- 强调风险
- 建议小额试水
- 推荐低风险产品
- 普及基础知识

进阶投资者内容：
- 深度分析
- 策略建议
- 多元配置
- 风险管理
```

### 原则3：风险揭示原则

**要求**：充分、明确、醒目地揭示风险

```
风险提示必须包含：

1. 市场风险
"投资有风险，入市需谨慎"

2. 产品风险
"基金过往业绩不代表未来表现"

3. 操作风险
"定投也需要择时和止盈"

4. 免责声明
"本文仅供参考，不构成投资建议"
```

## 具体要求

### 1. 内容审核标准

**数据准确性**：
```
检查清单：
□ 数据来源是否权威（Wind、Choice、交易所）
□ 数据是否最新（标注时间）
□ 数据计算是否正确
□ 对比基期是否合理
□ 引用是否准确无误
```

**观点合理性**：
```
检查清单：
□ 是否有充分论据支撑
□ 逻辑是否严密
□ 是否过度乐观或悲观
□ 是否考虑了反面情况
□ 是否留有余地（使用"可能"、"预计"）
```

**表达合规性**：
```
检查清单：
□ 是否有收益承诺（禁止！）
□ 是否有保本承诺（禁止！）
□ 是否有绝对化表述（避免）
□ 是否暗示无风险（禁止！）
□ 是否充分揭示风险
```

### 2. 禁止性条款

```
绝对禁止的内容：

❌ 收益承诺
"这款基金年化收益至少20%"
"跟着我买，稳赚不赔"

❌ 保本承诺
"本金绝对安全"
"不会亏钱的"

❌ 内幕消息
"我有内部消息"
"提前告诉你"

❌ 误导性陈述
"这是最好的产品"
"没有任何风险"

❌ 恶意竞争
"其他公司产品都不行"
"只有我们的最好"
```

### 3. 风险提示模板

**通用风险提示**：

```
【风险提示】

1. 市场风险
证券市场受经济政策、宏观经济、
投资心理和交易制度等各种因素影响，
可能导致投资收益波动。

2. 产品风险
不同产品有不同的风险特征，
请根据自身风险承受能力选择。

3. 流动性风险
部分产品可能存在流动性限制，
提前退出可能面临损失。

4. 免责声明
本文仅代表个人观点，
不构成投资建议。
投资者据此操作，风险自担。
```

**产品特定风险**：

```
【基金投资风险提示】

1. 净值波动风险
基金净值会随市场波动，
过去业绩不代表未来表现。

2. 流动性风险
开放日可能面临大额赎回，
影响基金运作。

3. 管理风险
基金经理变更、投资策略调整
可能影响基金表现。

4. 适当性提醒
请根据自身风险承受能力，
选择适合的基金产品。
```

## AI创作风险控制

### 1. AI幻觉应对

**识别AI幻觉**：

```
警示信号：
⚠️ 数据过于"完美"（如增长率正好是整数）
⚠️ 引用模糊（"某专家"、"某机构"）
⚠️ 逻辑跳跃（缺少中间环节）
⚠️ 与常识相悖
```

**核验流程**：

```
第一步：交叉验证
- 用多个来源对比数据
- 用历史数据验证逻辑
- 用常识判断合理性

第二步：权威确认
- 查阅官方数据（交易所、公司公告）
- 参考权威媒体（财联社、证券时报）
- 咨询专业人士

第三步：标注说明
- 数据标注来源
- 观点标注依据
- 不确定的内容使用"据"、"可能"
```

### 2. 内容合规检查

**检查清单**：

```
发布前必查：

□ 事实核查
  所有数据都有来源？
  所有引述都准确？
  所有观点都有依据？

□ 合规审查
  有无收益承诺？
  有无保本承诺？
  风险提示是否充分？
  免责声明是否完整？

□ 质量审核
  逻辑是否严密？
  表达是否准确？
  格式是否规范？
  错别字是否修正？

□ 留痕存档
  核验记录是否保存？
  修改记录是否留存？
  发布记录是否备份？
```

### 3. 责任划分

```
【AI职责】
- 资料检索整理
- 初稿内容生成
- 格式规范处理

【人工职责】
- 核心观点确定
- 事实数据核验
- 合规风险审查
- 最终内容把关

【责任原则】
AI是助手，不是决策者
最终责任始终在人工
```

## 特殊场景合规

### 1. 产品推荐

```
合规要求：

1. 适当性匹配
   评估客户风险承受能力
   推荐匹配风险等级的产品
   不推荐超出承受能力的产品

2. 信息披露
   充分揭示产品风险
   说明费用结构
   不夸大收益

3. 利益冲突
   如有利益冲突必须披露
   不为佣金推荐不合适产品
   客户利益优先

4. 留痕记录
   推荐理由记录
   风险告知记录
   客户确认记录
```

### 2. 市场评论

```
合规要求：

1. 观点客观
   基于事实和分析
   不主观臆断
   不散布谣言

2. 风险提示
   市场有风险
   观点仅供参考
   不构成投资建议

3. 信息来源
   引用权威来源
   不使用内幕信息
   不传播未证实消息

4. 谨慎预测
   使用"可能"、"预计"
   不做确定性判断
   提醒预测不确定性
```

### 3. 投教内容

```
合规要求：

1. 内容准确
   概念解释正确
   案例数据真实
   不误导投资者

2. 风险教育
   强调投资风险
   不美化投资收益
   培养风险意识

3. 实用性
   提供可操作建议
   结合实际情况
   不空谈理论

4. 更新及时
   跟进法规变化
   更新产品信息
   修正过时内容
```

## 典型违规案例

### 案例1：收益承诺

```
违规内容：
"这款基金年化收益至少20%，
跟着我买保证赚钱！"

正确表述：
"该基金过去3年年化收益率20%，
但过去业绩不代表未来表现。
投资有风险，需谨慎决策。"
```

### 案例2：风险提示不足

```
违规内容：
（全文没有风险提示，只讲收益）

正确表述：
（在内容中充分揭示风险）
"需要注意的是，
该产品波动较大，
适合风险承受能力较强的投资者。
请根据自身情况谨慎选择。"
```

### 案例3：使用未证实数据

```
违规内容：
"据消息，某公司将被某科技巨头收购。"

正确表述：
"市场有相关传闻，
但未获得官方确认。
请以公司公告为准，
不要盲目跟风。"
```

### 案例4：误导性表述

```
违规内容：
"这是目前最好的基金，
没有任何风险，稳赚不赔。"

正确表述：
"该基金在同类产品中表现较好，
但投资仍存在风险，
需要根据自身情况选择。"
```

## 合规工具和流程

### 1. 内容审核工具

```
合规检查工具包：

□ 关键词过滤
  - 过滤违规词汇
  - 标注风险表述
  - 提醒修改建议

□ 风险提示生成
  - 根据内容自动生成
  - 确保完整性
  - 标准化格式

□ 数据核验辅助
  - 标注需要核验的数据
  - 提示数据来源
  - 记录核验结果
```

### 2. 发布流程

```
内容发布标准流程：

1. AI生成初稿
   ↓
2. 事实核验
   - 数据准确性
   - 引用可靠性
   - 逻辑严密性
   ↓
3. 合规审查
   - 风险揭示
   - 禁止性条款
   - 免责声明
   ↓
4. 质量审核
   - 内容质量
   - 格式规范
   - 错误修正
   ↓
5. 留痕存档
   - 保存核验记录
   - 记录修改过程
   - 备份最终版本
   ↓
6. 发布
```

### 3. 应急处理

```
发现内容问题后的处理：

1. 立即撤回
   如发现错误或违规，
   立即撤回已发布内容。

2. 公开更正
   及时发布更正说明，
   解释错误原因，
   提供正确信息。

3. 内部复盘
   分析问题原因，
   完善审核流程，
   防止再次发生。

4. 责任追究
   明确责任归属，
   完善管理制度，
   加强培训教育。
```

## 总结

**核心理念**：
- 合规是底线，不可逾越
- 风险揭示是义务，必须履行
- 投资者保护是责任，时刻铭记

**关键要点**：
1. 真实性：所有信息必须真实可靠
2. 适当性：内容适合目标投资者
3. 风险揭示：充分、明确、醒目
4. 人工把关：AI辅助，人工负责
5. 留痕存档：全程记录，有据可查

**牢记**：
AI可以提高效率，
但合规责任始终在人工。
技术再进步，
合规底线不可破！
