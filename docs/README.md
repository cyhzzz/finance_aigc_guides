## 使用方式

### 更新操作指引内容

只需修改对应的MD文件即可：

```bash
# 编辑MD文件
docs/01-元宝分组配置.md
docs/02-知识库搭建.md
# ... 等等
```

修改后刷新页面即可看到更新。

### 添加新的操作指引

1. 在 `docs/` 文件夹创建新的MD文件
2. 在 `guides.html` 中添加新的卡片：

```html
<article class="card" onclick="openModal('文件名', '标题')" data-md="docs/文件名.md">
    <div class="card-icon">
        <i data-lucide="图标名" class="w-6 h-6" style="color: var(--stone)"></i>
    </div>
    <h3>卡片标题</h3>
    <p>卡片描述</p>
</article>
```

## 技术实现

### 使用的技术

- **Marked.js**：Markdown解析库
- **Lucide Icons**：图标库
- **原生JavaScript**：模态框交互逻辑
- **CSS动画**：平滑的过渡效果

### 性能优化

- **内容缓存**：已加载的MD文件会缓存，避免重复请求
- **懒加载**：只在点击时加载对应内容
- **CSS压缩**：内联关键CSS，减少请求
